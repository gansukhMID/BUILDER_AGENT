# Task: Core Mixins

## Description

Implement the three mixin classes that all models compose. These provide timestamp tracking, soft-delete, and name/code fields.

Files to create/modify:
- TEMPLATE_CORE/WMS/wms_core/mixins.py

Mixins to implement:
- `TimestampMixin`: `created_at: Mapped[datetime]` (server_default=now), `updated_at: Mapped[datetime]` (onupdate=now)
- `ActiveMixin`: `active: Mapped[bool]` (default=True); soft-delete pattern
- `NameMixin`: `name: Mapped[str]` (non-nullable), `code: Mapped[Optional[str]]` (nullable, unique per model)

All mixins use `MappedColumn` / `mapped_column` from SQLAlchemy 2.x. No `__tablename__` — these are pure mixin classes, not mapped models.

## Acceptance Criteria
- [ ] `from wms_core.mixins import TimestampMixin, ActiveMixin, NameMixin` succeeds
- [ ] A model class that inherits `TimestampMixin` gets `created_at` and `updated_at` auto-populated on insert
- [ ] `updated_at` updates on every flush
- [ ] `ActiveMixin.active` defaults to `True`
- [ ] `NameMixin.name` is non-nullable; `NameMixin.code` is nullable

## Technical Details

Use `datetime.utcnow` for default/onupdate (standard SQLAlchemy approach). Mixins must not import from any wms_core model file — only from `sqlalchemy` and `typing`.
