# Task: Ecommerce + User Pages + Seed Script + Backend Tests

## Description

Implement Ecommerce and User management frontend pages, create the backend seed script, and write the backend test suite.

**Ecommerce pages:**
- `/ecommerce` — Product catalog: grid of ProductTemplate cards with name, list_price, variant count
- `/ecommerce/orders` — Order list: table with reference, partner, state badge, total
- `/ecommerce/orders/[id]` — Order detail: header, line items table, payment state badge
- `/ecommerce/coupons` — Coupon list: code, discount, usage bar

**User pages:**
- `/users` — User table: email, membership tier badge, last login, active status
- `/users/[id]` — User profile: profile card (avatar placeholder, display name, bio), membership card, activity log timeline (last 20 events)

**Seed script (`backend/seed.py`):**
- 2 warehouses (each with 3 locations: Stock, Input, Output)
- 5 products (2 lot-tracked, 1 serial, 2 untracked)
- 10 quant records
- 3 users (1 admin, 2 regular) with profiles and memberships (1 VIP, 1 Pro, 1 Free)
- 5 sale orders with 2-3 lines each; mix of states
- 3 coupons
- 10 activity events

**Backend tests (`tests/`):**
- `test_auth.py`: register, login success, login wrong password, me endpoint
- `test_wms.py`: GET warehouses, locations, inventory, pickings, picking detail
- `test_ecommerce.py`: GET products, product detail, orders, order detail, stats
- `test_users.py`: GET users list, user detail, activity, user orders
- Coverage ≥80%

## Acceptance Criteria
- [ ] All ecommerce pages render with seeded data
- [ ] All user pages render with seeded data
- [ ] `/dashboard` stat cards show non-zero values after seed
- [ ] `python seed.py` runs without errors and populates all tables
- [ ] `pytest tests/ --cov=. --cov-report=term-missing` passes with ≥80% coverage
- [ ] `npm run build` exits 0

## Technical Details

Seed script must be idempotent (check if data exists before inserting). Tests use pytest with httpx `TestClient`. conftest.py creates a fresh in-memory SQLite DB per test session with all tables + seed data.
