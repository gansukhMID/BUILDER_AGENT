# Task: Package Scaffold

## Description

Create the full package skeleton for `TEMPLATE_CORE/ECOMMERCE`. This is the foundation all other tasks build on. Must match `TEMPLATE_CORE/WMS` conventions exactly (same `db.py` interface, same mixin class names, same `utils/state_machine.py` API).

## Acceptance Criteria
- [ ] `TEMPLATE_CORE/ECOMMERCE/` directory created with all skeleton files
- [ ] `ecommerce_core/db.py` exports `Base`, `engine_factory(dsn)`, `session_factory()`
- [ ] `ecommerce_core/mixins.py` exports `TimestampMixin`, `ActiveMixin`, `NameMixin`, `CurrencyMixin`
- [ ] `ecommerce_core/utils/state_machine.py` exports a generic FSM class with same API as `TEMPLATE_CORE/WMS/wms_core/utils/state_machine.py`
- [ ] `ecommerce_core/utils/pricing.py` placeholder created (empty `evaluate_pricelist` stub)
- [ ] `ecommerce_core/models/__init__.py` created (empty, populated in task 009)
- [ ] `pyproject.toml` with correct deps: `sqlalchemy>=2.0`, `alembic>=1.13`, `python-dotenv>=1.0`; dev: `pytest`, `pytest-cov`
- [ ] `alembic.ini.template` with `DATABASE_URL` placeholder DSN
- [ ] `alembic/env.py` reads `DATABASE_URL` from `.env` and imports `Base` from `ecommerce_core.db`
- [ ] `python -c "from ecommerce_core.db import Base, engine_factory, session_factory"` succeeds

## Technical Details

Mirror `TEMPLATE_CORE/WMS` package layout:
- `db.py`: `Base = declarative_base()`, `engine_factory(dsn: str) -> Engine`, `session_factory(engine) -> sessionmaker`
- `mixins.py`: `TimestampMixin` (`created_at`, `updated_at` via `__init__` or `@declared_attr`), `ActiveMixin` (`active: bool = True`), `NameMixin` (`name: str`, optional `code: str`), `CurrencyMixin` (`currency: str = 'USD'`)
- `utils/state_machine.py`: generic `StateMachine(model, field, transitions)` — same class signature as WMS
