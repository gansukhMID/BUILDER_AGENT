## Technical Details

`__table_args__` for abstract classes with Index/CheckConstraint: use string column names, not `cls.column` references, because the table doesn't exist yet:
```python
__table_args__ = (
    CheckConstraint("rating BETWEEN 1 AND 5", name="ck_review_rating"),
    Index("ix_review_target", "target_type", "target_id"),
)
```

`metadata_` naming: the trailing underscore is intentional — avoids shadowing Python's `metadata` keyword. Document this in the field comment.

`AbstractActivityEvent` is append-only by design — no `updated_at`. Log entries must never be mutated.
