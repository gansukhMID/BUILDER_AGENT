## Technical Details

Per AD-3, `user_id` on `AbstractOAuthAccount` is `Mapped[int] = mapped_column(nullable=False)` — no `ForeignKey()` in the abstract base. The agent adds the FK when subclassing:
```python
class OAuthAccount(AbstractOAuthAccount):
    __tablename__ = "oauth_account"
    user_id: Mapped[int] = mapped_column(ForeignKey("user.id"))
```

`set_password` / `check_password` stubs:
```python
def set_password(self, raw: str) -> None:
    """Override to hash raw using bcrypt/argon2. Stub sets nothing."""
    pass

def check_password(self, raw: str) -> bool:
    """Override to verify raw against hash. Stub always returns False."""
    return False
```

Use `from __future__ import annotations` and `__abstract__ = True` at class level.
