# Task: Frontend — Cart context

## Description

Create a React context that manages cart state, persisting to localStorage. This is a pure client-side module with no API dependency — it can be built before the store router exists.

Files to create:
- `APPS/odoo-frontend/frontend/lib/cart.tsx`

## Acceptance Criteria
- [ ] `CartContext` provides cart state to the `(store)` layout
- [ ] `useCart()` hook exposes: `items`, `addItem`, `removeItem`, `updateQty`, `clearCart`, `totalItems`, `totalPrice`
- [ ] Cart persists to localStorage key `store_cart` — survives page refresh
- [ ] Adding same variant increments qty rather than duplicating
- [ ] `totalPrice` applies correct arithmetic for all items
- [ ] TypeScript strict — no `any` types

## Technical Details

```typescript
// lib/cart.tsx
export interface CartItem {
  variantId: number;
  variantSku: string | null;
  productName: string;
  unitPrice: number;
  qty: number;
}

interface CartContextType {
  items: CartItem[];
  addItem: (item: Omit<CartItem, 'qty'>) => void;
  removeItem: (variantId: number) => void;
  updateQty: (variantId: number, qty: number) => void;
  clearCart: () => void;
  totalItems: number;
  totalPrice: number;
}
```

- Use `useEffect` to load from localStorage on mount
- Use `useEffect` to save to localStorage on every change
- SSR-safe: check `typeof window !== 'undefined'` before accessing localStorage
